API World of Warcraft

  • Lien EndPoint Créature: https://us.api.blizzard.com/data/wow/creature-family/index?namespace=static-us&locale=en_US
  • Lien EndPoint Sort : https://us.api.blizzard.com/data/wow/search/spell?namespace=static-us&name.en_US=Holy%20Shield&orderby=id
  • Documentation officielle : https://develop.battle.net/documentation/world-of-warcraft

Il faut savoir que cette API est assez spécial car les EndPoint ci-dessus font références à d'autres EndPoint qui me permettent soit d'avoir plus d'informations sur un objet en particulier soit de récupérer son image par exemple.

Mon site va donc utiliser dans un premier temps le EndPoint ci-dessus mais va en générer 1 voir 2 de plus dans mon script afin d'avoir accès à pus d'informations.

API Hearthstone

  • Lien EndPoint Cartes : https://eu.api.blizzard.com/hearthstone/cards
  • Lien EndPoint Dos de Cartes : https://eu.api.blizzard.com/hearthstone/cardbacks
  • Documentation officielle : https://develop.battle.net/documentation/hearthstone

Cette API provient de la même source que le premier cependant celui-ci est fait différement, pas besoin de passer par d'autre EndPoint comme le premier, toutes les infos ce trouve sur un unique Endpoint pour la catégorie en question.
